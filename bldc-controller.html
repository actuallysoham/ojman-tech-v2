BLDC Controller

Description

The BLDC Motor Controller can be customized for 48V to 72V input voltage designed for high-current, high power e-bike and e-rickshaw applications. This design uses efficient PWM techniques to provide best in class power conversion efficiency (>97%). The control algorithm has special feature for supporting flying start and regenerative braking

Technical Specifications

  • Up to 3500-W Continuous Power Stage: Bi-Directional Power Flow, to support regenerative braking
  • Power Conversion Efficiency: >97%
  • Up to 5000W peak power
  • Protections: Over Current, Over Voltage, Over-temperature
  • Software: High Performance Speed /Torque Control with Speed
  • Reversal Capability; Regenerative Braking, Flying Start Capability
